## Tally Form 설정

### 1. 제출 후 리다이렉트 설정

**리다이렉트 URL:**
```
https://snapplug.app/ai-diagnosis/result
```

**URL 파라미터 매핑:**
```
?role={Q1-A 응답}
&tasks={Q1-B 응답들, 쉼표로 구분}
&hours={Q2 응답}
&pattern={Q3 응답}
&team_size={Q4 응답}
&name={성함}
&company={회사명}
```

**예시 리다이렉트 URL:**
```
https://snapplug.app/ai-diagnosis/result?role=cs_담당자&tasks=고객_문의_응대,데이터_입력_정리&hours=10-20시간&pattern=대부분반복&team_size=6-20명&name=홍길동&company=ABC회사
```

### 2. Webhook 설정 (n8n 연동용)

**Webhook URL:** `[n8n Webhook URL 입력]`

## Tally Form 생성 단계

1. **Tally 접속:** https://tally.so
2. **새 폼 생성:** "Create new form"
3. **질문 추가:** 위 Q1-A ~ Q5 순서대로 추가
4. **설정:**
   - Form Settings → After submission → Redirect to URL
   - URL에 파라미터 매핑 설정
5. **Integrations:**
   - Webhook 연결 (n8n)
6. **퍼블리시:** 폼 URL 생성
