## 리다이렉트 URL (Settings → After submission)

```
https://snapplug.app/ai-diagnosis/result
```

### Hidden Fields 또는 URL Parameters 설정:

Tally에서 "Redirect to URL with form data" 옵션 선택 후,
각 질문의 값이 URL 파라미터로 전달되도록 설정

**파라미터 매핑:**
- Q1-A → `role`
- Q1-B → `tasks`
- Q2 → `hours`
- Q3 → `pattern`
- Q4 → `team_size`
- 성함 → `name`
- 회사명 → `company`
